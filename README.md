# Ternary Capital — Agents

Cognitive financial agents for investment research, portfolio monitoring, and thesis management.
Part of Ternary's open-source stack for agentic financial intelligence.

## What this is

This repository contains the agent layer of Ternary's cognitive financial intelligence stack. Agents here are built to operate with persistent context, grounded reasoning, and the reliability required by professionals who cannot afford to be wrong.

## Capabilities

- **Research** — screen tickers, underwrite fundamentals, detect variant perception
- **Portfolio management** — monitor positions, calibrate conviction, optimise construction
- **Risk** — stress-test theses, trigger capital preservation protocols, allocate through crises
- **Reporting** — generate institutional research reports, attribute performance, surface decision errors

## Architecture

Agents follow Ternary's four-property model:

1. **Market context** — grounded in live financial data through a dedicated world model
2. **Persistent memory** — intent-aware, tiered memory via DeepCQ
3. **Planning and execution** — can research, reason, and act towards a goal
4. **Self-correction** — reliable, controllable, and safe under uncertainty
